Questions fréquentes

Tout ce que vous devez savoir sur FocusBlocks.

Oui, FocusBlocks propose un pack Free avec 50 sessions par mois. Vous pouvez passer aux packs Premium pour débloquer plus de fonctionnalités.

L'application utilise les statistiques d'utilisation Android pour détecter quand vous changez d'application pendant une session. Elle classe ensuite les applications en distractives, neutres ou productives.

Après 5 sessions complétées, le Mode Intelligent s'active automatiquement. Il analyse vos habitudes et bloque ciblément les applications qui vous distraient le plus.

Absolument. Toutes vos données sont stockées de manière sécurisée via Firebase. Aucune donnée personnelle n'est partagée avec des tiers.

Pour le moment, FocusBlocks est disponible uniquement sur Android. La version iOS est en cours de développement.
